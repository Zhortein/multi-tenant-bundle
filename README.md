# Zhortein MultiTenantBundle

A Symfony 7+ bundle for clean, flexible and modular multi-tenancy.

## Features (MVP)
- Tenant resolution (subdomain, path, custom)
- Tenant context
- Doctrine tenant filter
- Commands
- Ready for storage/mailer/messenger per tenant
